# SaaS Customer Analytics using SQL & Python

## Project Overview
This project simulates a SaaS business environment by analyzing subscription-based customer data to understand revenue performance, customer segmentation, and churn behavior.

The objective was to perform end-to-end analytics using advanced SQL and Python.

## Tools & Technologies
- SQL (SQLite)
- Python (Pandas, Matplotlib)
- Jupyter Notebook

## Key SQL Concepts Used
- Joins
- Group By & Having
- Window Functions (NTILE, RANK)
- CTEs (WITH clause)
- Conditional Aggregation
- SaaS KPI calculations

## Business KPIs Analyzed
- Monthly Recurring Revenue (MRR)
- Customer Churn Rate
- Revenue Segmentation
- High-value customer identification

## Key Insights
- High-revenue customers showed lower churn probability.
- Month-to-month contracts contributed highest churn.
- Revenue concentration observed within top customer quartile.
